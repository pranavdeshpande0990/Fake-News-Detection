# TruthLens — Fake News Detection System
**Final Year Project | B.E. / B.Tech Computer Science**

---

## 📌 Project Overview

TruthLens is an AI-powered real-time fake news detection system that uses an ensemble of machine learning models trained on 45,000+ news articles. It achieves ~98% accuracy and provides confidence scores, linguistic signal analysis, word clouds, and a full analytics dashboard.

---

## 🏗️ Project Architecture

```
User Input (News Text)
        │
        ▼
┌───────────────────┐
│  Text Preprocessing│  ← lowercase, remove URLs, strip punctuation,
│                   │    remove stopwords
└─────────┬─────────┘
          │
          ▼
┌───────────────────┐
│  TF-IDF Vectorizer│  ← 10,000 features, unigrams + bigrams
│  (fit on training)│
└─────────┬─────────┘
          │
          ▼
┌─────────────────────────────────┐
│       Ensemble Classifier        │
│  ┌──────────┐ ┌───┐ ┌────────┐ │
│  │  Logistic│ │ NB│ │ Random │ │
│  │Regression│ │   │ │ Forest │ │
│  │ weight=3 │ │ 1 │ │  w=2   │ │
│  └────┬─────┘ └─┬─┘ └───┬────┘ │
│       └────┬────┘        │      │
│            └──── Soft Voting ───┘
└─────────────────┬───────────────┘
                  │
                  ▼
        Prediction + Confidence Score
        (FAKE / REAL with probability)
```

## 🚀 Setup & Installation

### Prerequisites
- Python 3.8+
- pip

### Install Dependencies

```bash
pip install streamlit pandas numpy scikit-learn nltk joblib \
            matplotlib seaborn plotly wordcloud
```

### Step 1 — Train the Model

```bash
python model.py
```

This will:
- Load and preprocess the dataset
- Train Logistic Regression, Naive Bayes, and Random Forest
- Build an ensemble with soft voting
- Evaluate and print accuracy, F1, AUC, classification report
- Save `classifier.pkl` and `vectorizer.pkl`
- Generate `model_evaluation.png`

### Step 2 — Run the Web App

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

## 📊 Model Performance

| Model | Accuracy | F1 Score | ROC-AUC |
|---|---|---|---|
| Logistic Regression | ~98.2% | ~0.982 | ~0.998 |
| Naive Bayes | ~94.1% | ~0.940 | ~0.987 |
| Random Forest | ~97.8% | ~0.978 | ~0.996 |
| **Ensemble (final)** | **~98.5%** | **~0.985** | **~0.999** |

## ✨ Features

### Core
- Real-time fake vs. real news classification
- Confidence probability scores (0–100%)
- Soft-voting ensemble model

### Analysis
- Linguistic signal breakdown (word count, exclamation marks, CAPS ratio)
- Word cloud visualization of key terms
- Probability bar chart for fake vs. real

### Dashboard
- Prediction history log
- Pie chart: fake vs. real distribution
- Histogram: confidence score distribution
- Timeline: cumulative prediction trend
- Clear history button

## 📚 Dataset

**ISOT Fake News Dataset** — University of Victoria
- **Fake.csv**: ~23,500 articles scraped from unreliable websites flagged by PolitiFact
- **True.csv**: ~21,400 articles from Reuters News Agency
- Both cover 2015–2018, primarily US political news
